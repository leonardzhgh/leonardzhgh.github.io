我们从这里开始...

近年来，快递业在国内迅速崛起，并且在国家的经济方面起到了一定的作用，但是伴随而来的确是堆积成山的快递箱。2018年，国家将垃圾分类列入工作中心之一。据不完全统计，2016年仅“双十一”期间就产生了15亿件包裹，但快递箱的回收利用不足30%。但是据我们调查，无论是损坏的还是没有损坏的、污染的还是没有污染的纸箱，都是可以进行重新打造成纸浆的，而且做出来的效果是一样的，成本也相同。那么为什么回收率还那么低呢？我们又该如何去提高回收率呢？本组织的成立致力于解决这一突出问题，帮助相关政府部门去解决这一大难题。

关于我们

RB ( Running Box ) 创立于2017年，主体是由中美国在校高中生成立并运营的学生组织。组织遵循联合国设立的可持续发展目标，主要致力于快递箱的回收环节的工作。 并且通过艺术展的宣传来帮助人们提高对于快递箱的关注度以及打造团队文化的知名度。与媒体以及其他的环保人士建立友好的关系。
